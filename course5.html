Course 5: Software Engineering

Course Description

This course provides an overview of software engineering principles and practices. Students will learn about the software development lifecycle, project management, and quality assurance.

Course Objectives

  • Understand the software development lifecycle.
  • Apply project management techniques in software development.
  • Implement quality assurance practices in software projects.
  • Utilize version control systems effectively.
  • Collaborate in a team environment using Agile methodologies.

Prerequisites

Completion of Course 4: Database Management Systems or equivalent experience is required.
