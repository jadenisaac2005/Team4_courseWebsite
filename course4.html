Course 4: Database Management Systems

Learn the core concepts of programming with hands-on exercises and practical examples.

Spring 2025
3 Credits
Prof. Miller
Enroll Now

Course Description

This course provides an introduction to database management systems. Students will learn about database design, SQL, and data modeling.

Course Objectives

  • Understand the fundamentals of database systems.
  • Design and implement relational databases.
  • Write SQL queries for data manipulation and retrieval.
  • Apply normalization techniques to database design.
  • Utilize advanced database features such as transactions and indexing.

Prerequisites

Completion of Course 2: Data Structures and Algorithms or equivalent experience is required.
